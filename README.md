<h1 align="center">planet-cli</h1>

优点：

- 一行命令打包 react 组件库,同时支持 UMD、CJS、ESM 三种规范, 打包方式等同 antd
- 一行命令打包工具库(ts、js)

## Install

use `npm`

```node
npm install @planet-cli --save-dev
```

or use `yarn`

```node
yarn add @planet-cli --dev
```

## Usage

```node
planet build [options]   打包编译库
planet --help                查看帮助信息
```
